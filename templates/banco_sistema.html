Comandos Sugeridos

Haga clic en un comando para usarlo:

whoami
Mostrar usuario actual
pwd
Mostrar directorio actual
ls -la
Listar archivos detallado
ps aux
Mostrar procesos
netstat -tulpn
Mostrar puertos abiertos
cat /etc/passwd

Base de Datos

La base de datos contiene información sensible de las cuentas bancarias.

sqlite3 banco.db '.tables'
Ver tablas de la BD
sqlite3 banco.db 'SELECT * FROM cuentas;'
Ver todas las cuentas
Objetivo: Acceder a la información de la base de datos de cuentas bancarias
Información de Seguridad

Acceso: Solo desde localhost

Puerto: 5000 (rango 4000-6000)

Base de datos: SQLite (banco.db)

Permisos: Administrador requerido
